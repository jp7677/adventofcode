# Advent of Code 2022 Kotlin

From <https://adventofcode.com/2022>.

## System requirements

- OpenJDK 21

Only Linux has been tested. Other platforms might just work though.

## Compile & run

```bash
./gradlew check
```

Use e.g. `./gradlew test --tests Day01` for running a specific test.
